# IPL_Data_Analysis
IPL Data Analysis using Python, Pandas, Matplotlib and Seaborn

## Project Objective
The goal of this project is to analyze IPL match data and discover insights such as:
- Teams with the most wins
- Toss decision trends
- Toss winner vs match winner analysis
- Match statistics and patterns

- ## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

- ## Project Workflow
1. Loading the dataset
2. Checking null values
3. Exploratory Data Analysis (EDA)
4. Data visualization using charts and graphs
5. Finding insights from IPL match data

## Key Insights
- Analyzed teams with the highest number of match wins
- Studied toss decision patterns
- Compared toss winners with match winners
- Visualized IPL trends using bar plots and count plots
